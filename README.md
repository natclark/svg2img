# svg2img

Simple SPA that `<svg>` elements to `<img>` elements.

I find it useful for automating some web accessibility related stuff.

*This repo is synced with Cloudflare Pages: https://svg2img.pages.dev*

## Setup

```
npm i
```

## Build

```
npm run build
```

## Run

```
npm run dev
```
